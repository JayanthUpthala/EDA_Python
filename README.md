# EDA Python - Wine Quality Analysis

This repository contains a Jupyter notebook for Exploratory Data Analysis (EDA) on the Wine Quality dataset.

## Dataset

The dataset used is `WineQT.csv`, which contains physicochemical properties of wine samples and their quality ratings. This dataset is sourced from Kaggle.

## Notebook Contents

The `eda-python.ipynb` notebook includes:

- Data loading and initial inspection
- Summary statistics
- Univariate analysis (e.g., distribution of quality ratings)
- Bivariate analysis and visualizations
- Correlation analysis

## Requirements

To run the notebook, you need:

- Python 3.x
- Jupyter Notebook or JupyterLab
- Required libraries: `numpy`, `pandas`, `matplotlib`, `seaborn`

You can install the dependencies using pip:

```
pip install numpy pandas matplotlib seaborn
```

## Usage

1. Clone this repository:
   ```
   git clone https://github.com/your-username/EDA_Python.git
   ```
2. Navigate to the directory:
   ```
   cd EDA_Python
   ```
3. Open the notebook:
   ```
   jupyter notebook eda-python.ipynb
   ```
4. Run the cells sequentially to perform the EDA.
